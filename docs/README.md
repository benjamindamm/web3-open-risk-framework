# 📚 Web3 Open Risk Framework Documentation

This directory contains the core documentation files that define the Web3 Open Risk Framework structure and methodology.

## 🗂️ Documentation Structure

### Core Framework Components

| File | Purpose | Description |
|------|---------|-------------|
| [`categories.md`](./categories.md) | **Risk Categories** | Defines the eight core risk categories (FINANCIAL, DEPENDENCY, TECHNICAL, etc.) |
| [`types.md`](./types.md) | **Risk Types** | Differentiates between QUANTITATIVE, QUALITATIVE, and HYBRID risks |
| [`scopes.md`](./scopes.md) | **Scope of Action** | Specifies where risks occur (TRADE, GOVERNANCE, APPLICATION, etc.) |
| [`objectives.md`](./objectives.md) | **Investment Objectives** | Maps risks to investment objectives (VALUE, YIELD, LIQUIDITY, etc.) |
| [`owners.md`](./owners.md) | **Risk Owners** | Defines the layered model of risk ownership and responsibilities |

### Templates and Guides

| File | Purpose | Description |
|------|---------|-------------|
| [`risk.md`](./risk.md) | **Risk Template** | Complete Markdown template for documenting individual risks |
| [`indicator.md`](./indicator.md) | **Indicator Template** | Defines how to create measurable metrics and behavioral signals |
| [`measure.md`](./measure.md) | **Measure Template** | Standardizes how to describe mitigation strategies |
| [`assessment.md`](./assessment.md) | **Assessment Guide** | Provides models for rating likelihood, impact, and persistence |

### Methodology and Philosophy

| File | Purpose | Description |
|------|---------|-------------|
| [`philosophy.md`](./philosophy.md) | **Framework Philosophy** | Describes methodology, principles, and external influences |
| [`terminology.md`](./terminology.md) | **Terminology** | Defines key terms and concepts for consistent usage |
| [`methodology.md`](./methodology.md) | **Methodology** | Detailed methodology for risk assessment and management |
